Over mij als developer

Ik ben een gemotiveerde student software development met veel interesse in webdesign, front-end en het bouwen van gebruiksvriendelijke websites.

Darvan Suleiman

Ik studeer software development op mbo 4 niveau en ik werk graag aan websites die modern, snel en overzichtelijk zijn. Wat mij aanspreekt in programmeren is dat je met code echt iets kunt opbouwen: van een idee naar een werkend product.

Mijn basis ligt sterk in HTML, CSS, JavaScript en Bootstrap. Daarnaast ben ik mezelf verder aan het ontwikkelen in Next.js, TypeScript, Prisma, Symfony, Tailwind CSS en SQL.
